# jenkinsvalidate

用来验证JenkinsFile  语法的正确性

## Features

在设置中输入jenkins的url 需要是host地址：
如：<http://localhost:8080/>

设置中输入jenkins的用户名和密码

通过在jenkinsfile 中使用command+p 然后输入validate jenkinsfile 即可
